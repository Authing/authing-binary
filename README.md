## Introduction
Authing-binary is a plug-in package of Guard-iOS.

Contains:
[Third-party identity source](https://docs.authing.cn/v2/reference/sdk-for-ios/social/)
[WebAuthn Biometric Authentication](https://docs.authing.cn/v2/reference/sdk-for-ios/webauthn.html)
